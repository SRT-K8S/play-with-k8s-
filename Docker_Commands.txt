>install Docker on our machine

	#yum install docker 

>know Docker version

	#docker version

>Searches the Docker Hub for images 
 
	#docker search

>list of containers 

	#docker ps

>Runs a command in a new container
 
	#docker run

>Starts one or more stopped containers

	#docker start

>Stops one or more running containers

	#docker stop

>Builds an image form a Docker file

	#docker build

>list of images 

	#docker Images

>Pulls an image or a repository from a registry

	#docker pull

>Pushes an image or a repository to a registry

	#docker push

>Remove one or more Images 

	#docker rmi

>Exports a container’s filesystem as a tar archive

	#docker export

>Runs a command in a run-time container

	#docker exec

>Attaches to a running container

	#docker attach

>Creates a new image from a container’s changes

	#docker commit

>Memory usage on our system 

	#docker stats

>Disk usage of docker 

	#docker system df

>Remove unused data

	#docker system prune
